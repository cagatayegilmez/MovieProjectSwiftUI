## Overview

MovieProjectSwiftUI is a feature-rich demo application built with SwiftUI, demonstrating modern iOS development best practices. It showcases a modular, testable architecture that integrates MVVM, Combine, async/await, and CoreData to create a responsive and scalable movie browsing experience.
